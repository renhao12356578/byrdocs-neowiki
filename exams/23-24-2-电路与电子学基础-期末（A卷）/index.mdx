## 四、计算题

<Figure src="题图4.svg" float>
    图4
</Figure>

电路如图4所示，已知 $u_s = 16 \sqrt{2} \cos 10^4 t$，请写出 $u_s$ 的有效值相量，并求ab两端的戴维南等效电路。

<Solution>
    $j \omega L = j 8 \Omega$

    $\frac{1}{j \omega C} = -j 4 \Omega$

    开路电压：$U_{oc} = \frac{j 8}{j 8 - j 4} U_s = 32 V$

    短路电流：$I_{sc} = j 4 = 4 \angle 90^\circ$

    等效电阻：$R_{eq} = \frac{U_{oc}}{I_{sc}} = \frac{32}{j 4} U_s = -j 8 \Omega$

    或 $R_{eq} = j 8 // -j 4 = -j 8 \Omega$
</Solution>

## 七、计算题

<Figure src="题图7.svg" float>

电路如图7所示，已知 $u_o = -55 u_I$，其余参数如图中所标注。试求出 $R_5$ 的值；并说明若 $u_I$ 与地接反，则输出电压与输入电压的放大倍数（比值）将产生什么变化。

<Solution>
    如图所示，$A_1$ 构成同相比例运算电路，$A_2$ 构成反相比例运算电路。

    $u_{01} = (1 + \frac{R_2}{R_1}) u_1 = (1 + \frac{100 k\Omega}{10 k\Omega}) u_1 = 11 u_1$

    $u_0 = -\frac{R_5}{R_4} u_{01} = -\frac{R_5}{100 k\Omega} \times 11 u_I = -55 u_I$

    $R_5 = 500 k\Omega$

    若 $u_I$ 与地接反，则第一级变为反相比例运算电路。

    $u_{01} = -\frac{R_2}{R_1} u_I = -\frac{100 k\Omega}{10 k\Omega} u_I = -10 u_I$

    由于第二级电路的比例系数仍为-5，所以输出电压与输入电压的比例系数变为50。
</Solution>
