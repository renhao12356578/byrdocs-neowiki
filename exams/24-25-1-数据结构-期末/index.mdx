## 四、程序设计（10分）

假设以带头结点的单链表表示有序表，单链表的类型定义如下：

```cpp
typedef struct node{
	DataType data;
	struct node *next;
}LinkNode, *LinkList;
```

试编写算法，从有序表 `A` 中删除所有和有序表 `B` 中元素相同的结点。

<Solution>
```cpp
void DeleteCommon(LinkList &A, LinkList B) {
    LinkList pa, pb, prev, tmp;
    pb = B->next;
    while (pb != NULL) {
        prev = A;
        pa = A->next;
        while (pa != NULL) {
            if (pa->data == pb->data) {
                prev->next = pa->next;
                tmp = pa;
                pa = pa->next;
                free(tmp);
            } else {
                prev = pa;
                pa = pa->next;
            }
        }
        pb = pb->next;
    }
}
```
</Solution>
